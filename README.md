# codex-skills

Repository for maintaining reusable Codex skills, supporting knowledge assets, and agent templates.

## Structure

- `skills/` stores individual skill definitions and related assets.
- `knowledge/` stores shared reference material used by skills or agents.
- `agents.md` provides a template for documenting agent roles, prompts, and operating rules.

## Getting Started

1. Add a skill under `skills/<skill-name>/`.
2. Place shared docs, notes, or references under `knowledge/`.
3. Copy sections from `agents.md` when defining a new agent.
